# Changelog

## 0.1.0 - 2026-06-09

### Added

- Initial Argus Delta Courtlens Vite/React/TypeScript MV3 extension.
- React Shadow DOM sidebar with Overview, Research, Documents, and Settings tabs.
- NSW court-list parsing, party candidate extraction, NSW Caselaw metadata extraction, and deterministic judgment entity extraction.
- Provider router for Argus Delta, Google News, Federal Court, NSW Caselaw, and ABN seam.
- Argus Delta bearer-token search support with quote stripping and nullable-result normalization.
- Document application payload builder and court-recipient routing.
- Copied PDF templates and pdf-lib/fontkit vendor assets from the mature NSW Courts+ extension.
- Vitest tests and smoke verification script.
